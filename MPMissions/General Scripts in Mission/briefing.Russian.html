Правила нажатия клавиш Shift-Alt-Click

Традиционно в CRCTI используется "Shift-Alt-Click" для открытия диалогового окна. В последней версии TZK этот клик по карте вызывает различные диалоги в зависимости от текущего состояния RTS игрока.

- Выберите нужный AI

Будет активирован диалоговый интерфейс команд "RTS". Также будет нарисована область, нажатие клавиши Enter быстро переключит на диалоговый интерфейс "Область".

- "Кликните без выбора AI, но с нарисованным прямоугольником.">

Будет активирован диалоговый интерфейс "Область".

- "Кликнуть без выбора AI и без нарисованного прямоугольника."

Будет активирован диалоговый интерфейс "POINT" или "WAYPOINT", в зависимости от того, какой был активирован в последний раз. Нажатие клавиши Enter позволит быстро переключиться на другой интерфейс.

Нажми ON/OFF режим RTS

В традиционной CRCTI используется "Alt-Click без AI" для быстрой установки точек пути игроков.В последней версии TZK также используется Alt-Click для рисования маркеров RTS. Активируйте радиоканал 0-0-1, чтобы переключаться между этими двумя режимами.

CTI (Capture the Island) - это режим стратегии в реальном времени для многопользовательских миссий в игре Operation Flashpoint. В режиме CTI игроки с разных сторон должны стратегически использовать свои ресурсы для строительства заводов, покупки боевых единиц, победы над вражескими силами и, в конечном итоге, завоевания острова для победы.


Objectives:
(Для победы необходимо завоевать или выполнить одну из целей.)

Уничтожить MHQ и постройки (Barracks, Light, Heavy and Air).

Взять все города (и держать 2минуты).

Командующий противника сдался.
